## Installation
1. Clone the repo
```sh
git clone https://github.com/KrzysztofBienias/the-abcde-ecommerce.git
```

2. Install packages
```sh
npm install
# or
yarn install
```

3. Create and fill `.env.local`
```sh
GOOGLE_ID=...
GOOGLE_SECRET=...

NEXTAUTH_SECRET=...

STRIPE_PUBLIC_KEY=...
STRIPE_SECRET_KEY=...
STRIPE_SIGNING_SECRET=...
```

4. Run the development server
```sh
npm run dev
# or
yarn dev
```
